# Povl-Api
Povl-Api项目目前用于熟悉SpringCloud微服务组件，以及分布式架构。
使用到的技术栈
Spring Boot + MySQL + Mybatis + Redis + Dubbo + Gateway + Ant Design Pro + Axios + TS
